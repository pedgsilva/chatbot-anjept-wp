# ChatBot ANJE - WordPress Plugin

Plugin WordPress para o chatbot da ANJE (anje.pt).

## Funcionalidades

- Chatbot inteligente integrado com OpenRouter (LLM)
- Nome do chatbot personalizável
- Cor principal e posição configuráveis
- Mensagem de boas-vindas personalizável
- Base de conhecimento completa da ANJE:
  - Órgãos sociais (Direção Nacional, Mesa da Assembleia-Geral, Conselho Fiscal)
  - Programas (Incubação, Formação, Prémio Jovem Empreendedor, MOVE)
  - Como se tornar associado
  - Benefícios de associado
  - Contactos e estatutos
- Suporte a backend Flask (proxy) ou API direta
- Design responsivo (mobile-friendly)
- Animações suaves
- Sem dependências externas

## Instalação

1. Fazer upload da pasta para `/wp-content/plugins/chatbot-anjept-wp/`
2. Ativar o plugin em **Plugins**
3. Configurar em **Definições > ChatBot ANJE**

## Configuração

1. Inserir a OpenRouter API Key
2. Personalizar nome, cor, mensagem de boas-vindas
3. Guardar
